Hashwords

Human-readable hashes for Node.js and the browser

Source on GitHub

Installation - Node.js


$ npm install hashwords

Installation - Browser


$ bower install hashwords

Bower is prefered, but you can also download a zip.

Usage


// Node.js:
// var hashwords = require('hashwords');

// Browser:
// <script src="path/to/hashwords.min.js"></script>
 
var hw = hashwords(),
    salt = 'saltysalt111',
    hwSalt = hashwords({salt: salt}),
    input = 'user@email.com';

hw.hash(input);
// ['Political', 'Predator', 'Project'] 

hw.hashStr(input);
// 'Political Predator Project'

hwSalt.hashStr(input);
// 'Shoddy Pond Installation'

hw.random();
// Something like
// ['Enormous', 'Baby', 'Slice']

hw.randomStr();
// Something like
// 'Plain Shirt Facility'
